# Reverse Words In A String

- LeetCode No: 151
- Problem: https://leetcode.com/problems/reverse-words-in-a-string/
- Study Plan: https://leetcode.com/studyplan/leetcode-75/
- Status: Done
- Approach Used: Split + Manual Reverse

## Approach

Use split and manual reversing.

- Split the string into a list of words using `split()`.
- Reverse the list by swapping words from the beginning and end.
- Join the reversed words back into one string with a single space.

## Complexity

- Time: `O(n)`
- Space: `O(n)`
